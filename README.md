# 💬 Chat Application

A real-time chat application developed using Flutter and Firebase. The application allows users to register, log in securely, and communicate through instant messaging with a clean and user-friendly interface.

## ✨ Features

- 🔐 User Authentication
- 💬 Real-Time Messaging
- 👤 User Profile
- 🎨 Theme Changing
- ⚙️ Settings Page
- 🚀 Firebase Integration

## 🛠️ Technologies Used

- Flutter
- Dart
- Firebase Authentication
- Cloud Firestore
- Android Studio
